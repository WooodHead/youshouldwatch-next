# You Should Watch

An experimental app to try some new stuff

## The Movie DB

You need an API KEY from
https://developers.themoviedb.org/3/getting-started/introduction

Create a `.env` file that looks like this:

```sh
THEMOVIEDB_API_KEY=a807c9..........
```

## Quickstart

Run:

```sh
pnpm install
pnpm run dev
```

To test, run:

```sh
pnpm run lint
pnpm run build
```
